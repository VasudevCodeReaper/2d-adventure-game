# 2D Adventure Game

A simple 2D platformer game built using the Godot Engine.  
The game focuses on platform movement, physics, enemy interaction, and coin collection mechanics.

---

## Features

- 2D platformer movement
- Platform physics
- Enemy system
- Coin collection system
- Global coin counter
- Win condition after collecting all coins
- Scene-based level design
- Organized asset and script structure

---

## Gameplay

The player explores the level while avoiding enemies and collecting coins.

Each collected coin increases the global coin count.  
Once all coins in the level are collected, the player wins the game.

---

## Project Structure

```text
assets/     -> sprites, textures, sounds
scenes/     -> game scenes
scripts/    -> gameplay scripts
```

## How to Run

1. Install Godot Engine
2. Clone this repository:

```bash
git clone https://github.com/VasudevCodeReaper/2d-adventure-game.git
```

3. Open the project using `project.godot`
4. Run the main scene in Godot
